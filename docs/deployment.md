# Deployment & DevOps

## Frontend
- Use Docker for local/dev/prod
- Deploy to easypanel
- Use env variables from `env-frontend.example.md`
- Push to GitHub
- Use GitHub Actions for CI/CD

## Backend
- Use Docker for local/dev/prod
- Deploy to easypanel
- Use env variables from `env-backend.example.md`
- Database migration on start
- Push to GitHub
- Use GitHub Actions for CI/CD

## General
- All secrets in env vars
- Test all endpoints/pages before launch
- Monitor logs and events

See `frontend.md`, `backend.md`, and `env.example.md` for more.
